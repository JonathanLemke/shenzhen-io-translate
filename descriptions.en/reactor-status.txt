* *error* é uma entrada do XBus indicando o erro atual do reator, se houver.
* *power* é uma entrada simples que indica a saída de energia atual do gerador.
* *temp* é uma entrada simples que indica a temperatura atual do núcleo do reator.
* *control* é uma entrada simples que indica o estado atual das hastes de controle do reator.
* *display* é uma saída de vídeo XBus correspondente a um LCD personalizado LX900C.
* Atualize continuamente o *display* para mostrar o status atual do reator.

##############

* Para ativar um segmento *display*, envie o ID desse segmento para o *display* sobre o XBus. Para desativar um segmento *display*, envie o ID desse segmento multiplicado por -1.
* Para desligar todos os segmentos *display*, envie o valor -999 para o *display* sobre o XBus.
* Se a leitura de *error* fornecer um valor, envie esse valor para *display* inalterado para ativar o indicador *error* correspondente (positivo) ou desligado (negativo).
* Consulte a seção "Dados Suplementares" do manual para encontrar as ESPECIFICAÇÕES DO LCD PERSONALIZADO para o *display*, além de instruções específicas sobre como exibir as diferentes informações de status.
