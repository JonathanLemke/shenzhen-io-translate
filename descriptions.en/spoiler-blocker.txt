* *audio-in* é uma entrada simples conectada a um microfone omnidirecional.
* *audio-out* é uma saída simples conectada a um par de fones de ouvido.
* *keywords* é uma entrada XBus sem bloqueio conectada a um dispositivo de processamento de linguagem natural que detecta palavras faladas em um fluxo de áudio.
* *override* é uma entrada simples conectada a um botão.

##############

* O sinal de *audio-in* deve ser copiado para *audio-out* até que seja detectada uma palavra-chave da lista de PALAVRAS-CHAVE RELACIONADAS AO SPOILER, para que a aclamada série de televisão GET THE THRONE seja detectada, 
após o qual * audio-out * deve ser bloqueado para um valor de 50 até que o botão *override* seja pressionado.
* Consulte a seção "Dados Suplementares" do manual para encontrar as PALAVRAS-CHAVE RELACIONADAS COM SPOILER de David e Xiaomei para vários programas de televisão.
